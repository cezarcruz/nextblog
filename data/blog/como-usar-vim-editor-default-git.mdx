Quando instalamos o git pode ser que ele user o nano ou o vi como editor padrão para commits, eventualmente seu sistema pode não ter instalado ou você gostaria de usar outro editor padrão, para usar o vim como editor padrão, basta executar o seguinte comando:


```bash
git config --global core.editor "vim"
```
Lembrando que o Vim precisa estar instalado ;)
